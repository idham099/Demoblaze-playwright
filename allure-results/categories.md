# Allure Report Categories

## App Bugs (Defect)
Ini adalah kegagalan yang disebabkan oleh kesalahan di aplikasi yang sedang diuji (SUT). Contoh: Assertion UI gagal karena elemen tidak muncul atau teks salah.

## Environment Issues
Ini adalah kegagalan yang bukan disebabkan oleh kode test atau aplikasi, melainkan masalah di lingkungan eksekusi. Contoh: Network error, server offline, atau timeout koneksi.

## Test Design Issues
Ini adalah kegagalan yang disebabkan oleh kode automation yang buruk. Contoh: Locator salah, sinkronisasi yang buruk, atau kesalahan sintaksis.
